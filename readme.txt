Цель работы: реализовать раскатку Nginx с помощью средств автоматизации: Gitlab, Python, Ansible.
В процессе работы решены следующие задачи:
1. Развернут linux-сервер на виртуальной машине при помощи VirtualBox
2. На сервер установлены gitlab-сe, gitlab-runner
3. Произведена проверка подключения к машине через cmd windows, и проверка работы UI Gitlab
4. Настроено подключение VSCode к локальному проекту Gitlab
5. Настроена автоматическая раскатка Nginx с помощью Gitlab (shell)
6. Настроена автоматическая раскатка Nginx с помощью Python
7. Настроена автоматическая раскатка Nginx с помощью Ansible
8. Настроено автоматическое удаление Nginx 

Настройка автоматической раскатки средствами Gitlab:
- создан pipeline с jobs, отвечающими за каждую поставленную задачу, в том числе кнопка "approve", 
созданная для проверки обязательных условий выполнения jobs.
- установка и удаление Nginx здесь осуществляются с помощью команд linux shell, передаваемых на сервер.
- произведена подмена конфигурационного файла с помощью переменной Giltab, закодированной в base64

Настройка автоматической раскатки средствами Python:
- создан скрипт с подключением необходимых для управления linux библиотек, 
в котором осуществляется установка необходимых компонентов, замена конфигурационного файла, перезапуск сервиса и проверка статуса

Настройка автоматической раскатки средствами Ansible:
- создан основной сценарий, к нему: роли, переменные, среды. 
Осуществлен параметризованный deploy: в зависимости от выбранной среды осуществляется настройка Nginx на определенный порт.
